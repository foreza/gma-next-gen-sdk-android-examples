# Google Mobile Ads Next-Gen SDK for Android

The GMA Next-Gen SDK enables mobile app developers to maximize their
monetization in Android mobile apps.

This repository contains open source examples and developer resources for the
GMA Next-Gen SDK, covering both the Google AdMob and Google Ad Manager.

## Support

To get developer support, visit the [Mobile Ads SDK
Support](https://developers.google.com/admob/support) page.

## Community

Join our new [Discord channel](http://goo.gle/admob-discord) dedicated to
discussion of the GMA Next-Gen SDK, to chat with the team and other developers.

## Documentation

Check out our [Get
started](https://developers.google.com/admob/android/next-gen/quick-start) guide
with the GMA Next-Gen SDK.
